# Computational-Lithography-Simulator
This will be updated soon.

## Goal

This project aims to understand optical lithography and optical proximity correction by implementing the imaging process in Python.

## Motivation

During my work in Samsung CIS PA, I handled GDS, e-beam inspection, and mask-related evaluation. While comparing design data with manufactured mask patterns, I became interested in understanding how optical imaging affects the final wafer pattern.

To deepen this understanding, I decided to implement a computational lithography simulator from scratch.
